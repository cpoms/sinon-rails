# Sinon.js for Rails

sinon.js via asset pipeline

## Installation

Add this line to your application's Gemfile:

    gem 'sinon-rails'

And then execute:

    $ bundle

Add this line to your test manifest file:

    //=require sinon

Or directly include at `/assets/sinon.js`

## Usage

Read about sinon.js [here](http://sinonjs.org/).
